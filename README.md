## Zappy - ZapSurveys Penetration Testing Tool

## How To Use

You need to install requirements.txt:

```python
pip install -r requirements.txt
pip3 install -r requirements.txt
```

Then simply run do:
```
python main.py 
python3 main.py
```

### Enjoy!
